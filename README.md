# Sales Data Dashboard

## Overview
This Power BI dashboard provides insights into sales performance across different customers. It helps analyze sales trends, identify top-performing customers, and track key business metrics through interactive visualizations.

## Features
- Customer-wise sales analysis
- Sales performance comparison
- Interactive charts and visuals
- KPI tracking for business insights
- Easy-to-understand dashboard design

## Dashboard Preview
![Sales Dashboard](dashboard.png)

## Tools Used
- Power BI
- Excel / CSV Dataset
- Data Modeling
- Data Visualization

## Key Insights
- Identifies customers with the highest sales contribution
- Compares sales performance across customer IDs
- Supports data-driven decision making
